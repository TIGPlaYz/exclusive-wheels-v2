Broken Siren Lights

  1. You need to Grab two meta files. carvariations.meta and carcols.meta,
  2. Open carcols.meta using Visual Studio Code or Notepad .
  3. Search for Sirens. Under Sirens you will Find id value="numbers"
  4. Click on the ID Value Number and Change it to any Random Numbers Ex-12534
  5. Open carvariations.meta using Visual Studio Code or Notepad .
  6. Search for sirensettings category and under that change the Value of sirensettings to the same number you changed in carcols.meta in this case it will be 12534.
